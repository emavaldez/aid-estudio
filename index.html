1. Preparación de Datos y Muestreo

📝 Resumen

La preparación de datos es una etapa fundamental del análisis de datos que incluye el análisis previo, tratamiento de valores perdidos, detección y manejo de casos atípicos, transformaciones y muestreo.

Valores Perdidos (Missing Values)

  • Son datos ausentes representados como NA, comunes en conjuntos de datos reales
  • Detección: comprobar existencia, cantidad y porcentaje de valores ausentes, por columna y fila
  • Tratamiento: rellenar con media/mediana/moda, forward/backward fill, eliminar filas, eliminar variables con >50% ausentes
  • La estrategia más común (valor medio) no siempre es la correcta; puede modificar resultados, reducir tamaño muestral o introducir sesgo

Casos Atípicos (Outliers)

  • Observaciones significativamente distintas del resto de datos
  • Detección en variables continuas: histogramas, boxplots (gráficos de cajas y bigotes)
  • Detección en variables categóricas: gráficos de barras observando categorías con pocas observaciones
  • Tratamiento: reducir influencia, no eliminar sin verificar. Repetir análisis con y sin outliers para observar efecto

Transformación de Datos

MétodoFórmulaCuándo usar
NormalizaciónX_norm = (X - X_min) / (X_max - X_min)Distribución no normal; escala [0,1]
EstandarizaciónZ = (X - μ) / σDistribución normal; media=0, DE=1
Escalamiento RobustoX_r = (X - Mediana) / RICMuchos outliers; usa Rango Intercuartílico

Técnicas de Muestreo

  • Aleatorio simple: cada elemento tiene igual probabilidad de ser elegido
  • Sistemático: se toma cada k elementos de una lista ordenada
  • Estratificado: se divide la población en estratos homogéneos y se muestrea dentro de cada uno
  • Con/Sin reemplazo: con reemplazo permite repetir elementos; sin reemplazo los elimina

Aplicaciones del Muestreo

  • Train/Test split: separar datos en entrenamiento y prueba (usualmente 70-30%)
  • Validación Cruzada (k-fold): dividir datos en k partes; entrena con k-1, valida con 1; repite k veces y promedia
  • Datos Desbalanceados: sobremuestreo de clase minoritaria, submuestreo de clase mayoritaria, generación sintética
  • Bootstrapping: remuestreo con reemplazo para estimar propiedades estadísticas
  • Permutaciones: reordenar etiquetas para construir distribución nula

Distribuciones clave: Bernoulli Binomial Poisson Normal Exponencial t de Student Chi-cuadrado F de Snedecor

Ley de los Grandes Números (LGN): la media muestral converge a la media poblacional al aumentar n.

Teorema Central del Límite (TCL): la distribución de las medias muestrales se aproxima a una normal, independientemente de la distribución original, para n grande.

2. Test de Hipótesis (Básico)

📝 Resumen

El test de hipótesis es una técnica estadística para tomar decisiones sobre parámetros poblacionales basándose en información muestral.

Estructura del Test

  • H₀ (Hipótesis Nula): la hipótesis que se contrasta (generalmente "no hay efecto/diferencia")
  • H₁ (Hipótesis Alternativa): la hipótesis que se investiga
  • Hipótesis paramétricas: se refieren al valor de un parámetro
  • No paramétricas: no se refieren al valor del parámetro

Nivel de Significación y Errores

H₀ verdaderaH₀ falsa
No rechazar H₀✅ Correcto (1-α)❌ Error Tipo II (β)
Rechazar H₀❌ Error Tipo I (α)✅ Potencia (1-β)
  • α (Error Tipo I): rechazar H₀ cuando es verdadera = "falso positivo"
  • β (Error Tipo II): no rechazar H₀ cuando es falsa = "falso negativo"
  • Potencia (1-β): probabilidad de rechazar correctamente H₀ cuando es falsa
  • Al aumentar n, se reduce β (mayor potencia)
  • Al disminuir α, aumenta β (y viceversa)

P-valor

  • Es la probabilidad de obtener un resultado al menos tan extremo como el observado, asumiendo H₀ verdadera
  • Criterio: rechazar H₀ si p-valor ≤ α
  • P-valor pequeño → fuerte evidencia contra H₀
  • P-valor grande → datos consistentes con H₀
  • Fisher propuso α = 0.05 como punto de corte

Tipos de Contraste

  • Unilateral: H₁ apunta en una dirección (μ > μ₀ o μ < μ₀)
  • Bilateral: H₁ apunta en ambas direcciones (μ ≠ μ₀)

Matriz de Confusión (Introducción)

Predicho 0Predicho 1
Real 0VN (Verdadero Negativo)FP (Falso Positivo)
Real 1FN (Falso Negativo)VP (Verdadero Positivo)
  • Accuracy: (VP + VN) / Total
  • Precisión: VP / (VP + FP)
  • Sensibilidad (Recall): VP / (VP + FN)
  • Especificidad: VN / (VN + FP)

Test Chi Cuadrado

  • Se usa para datos categóricos en tablas de doble entrada (r × c)
  • Test de Independencia: verifica si dos variables categóricas son independientes. Se selecciona una muestra n y se determina para cada sujeto el nivel de A y B
  • Test de Homogeneidad: verifica si la distribución de una variable es la misma en diferentes poblaciones
  • Frecuencias esperadas deben ser > 5 (máximo 20% con < 5)
  • Si no se cumple → Test exacto de Fisher

3. Distancia y Medidas de Proximidad

📝 Resumen

Las medidas de distancia cuantifican qué tan "lejos" están dos puntos en un espacio multidimensional. Son fundamentales en agrupamiento, clasificación y detección de anomalías.

Distancia Euclídea

La distancia en línea recta entre dos puntos, basada en el Teorema de Pitágoras.

d(A,B) = √[∑(bᵢ - aᵢ)²] para i = 1 a n
  • En 2D: d = √[(x₂-x₁)² + (y₂-y₁)²]
  • Es la más utilizada, sensible a la escala de las variables
  • Los puntos a distancia r conforman una circunferencia

Distancia de Manhattan (Taxista)

Suma de las diferencias absolutas entre coordenadas.

D(A,B) = ∑|bᵢ - aᵢ| para i = 1 a n
  • En 2D: D = |x₂-x₁| + |y₂-y₁|
  • Los puntos a distancia r conforman un rombo de diagonal 2r
  • Aplicaciones: k-means, reconocimiento de imágenes, detección de anomalías, SIG

Distancia de Chebyshev

Es la máxima diferencia absoluta entre las coordenadas.

D(A,B) = max|bᵢ - aᵢ|
  • Los puntos a distancia r conforman un cuadrado de lado 2r

Distancia de Minkowski

Generalización de las distancias anteriores con un parámetro p.

D(A,B) = [∑|bᵢ - aᵢ|ᵖ]^(1/p)
  • p = 1 → Manhattan
  • p = 2 → Euclídea
  • p → ∞ → Chebyshev

Medidas de Proximidad para Variables Categóricas

  • Coeficiente de Jaccard: mide la similitud entre conjuntos. J = |A∩B| / |A∪B|
  • Útil para variables binarias (presencia/ausencia)

4. Análisis de Componentes Principales (PCA)

📝 Resumen

El PCA es un método estadístico de reducción de dimensionalidad que transforma variables correlacionadas en nuevas variables no correlacionadas (componentes principales) que maximizan la varianza.

Conceptos Clave

  • Pertenece al aprendizaje no supervisado
  • Combina p variables originales en z componentes (z < p) que son combinaciones lineales de las originales
  • Las componentes son incorrelacionadas entre sí
  • Técnica descriptiva, libre de distribución, exploratoria (sin supuestos)

Matriz de Trabajo

  • Si variables en misma escala: Matriz de varianzas-covarianzas
  • Si variables en distintas escalas: Matriz de correlación (equivale a covarianza de variables estandarizadas)

Autovalores y Autovectores

  • Autovalores (λ): representan la varianza explicada por cada componente
  • Autovectores: direcciones de máxima varianza; son las cargas (loadings)
  • Se calculan resolviendo: Det(A - λI) = 0 (polinomio característico)
  • La traza = suma de autovalores = varianza total
  • El determinante = producto de autovalores

Cargas (Loadings)

  • Carga positiva → correlación positiva entre variable y componente
  • Carga negativa → correlación negativa
  • Magnitud alta → la variable contribuye mucho a esa componente

Proporción de Varianza Explicada

% Varianza = λᵢ / ∑λⱼ × 100

Se seleccionan las componentes que explican un porcentaje acumulado deseado (usualmente 70-80%).

Biplot

  • Grafica simultáneamente individuos y variables
  • Facilita: interpretación de distancias, búsqueda de grupos, explicación de componentes
  • Útil cuando hay pocas observaciones

Relaciones Lineales

  • Dependencia lineal: α₁v₁ + α₂v₂ + ... + αₙvₙ = 0 con algún αᵢ ≠ 0 (información redundante)
  • Independencia lineal: solo si todos los escalares son cero

5. PCA Robusta

📝 Resumen

El PCA clásico es sensible a outliers. Los métodos robustos reemplazan la media y la matriz de covarianza clásica por estimadores resistentes a valores atípicos.

Problema del PCA Clásico

  • Los outliers univariados o multivariados pueden distorsionar la matriz de covarianza muestral
  • Conducen a resultados erróneas en las componentes principales

Alternativas Robustas

MétodoDescripción
MCD (Minimum Covariance Determinant)Selecciona submuestra con menor determinante de covarianza. h = [(n+p+1)/2]
MVE (Minimum Volume Ellipsoid)Busca el elipsoide de volumen mínimo que contiene al menos la mitad de los puntos

MCD en Detalle

  • Busca un subconjunto de h observaciones cuya matriz de covarianza tenga el menor determinante
  • Algoritmo rápido de Rousseeuw y Van Driessen
  • Converge en número finito de pasos a un mínimo local
  • El estimador MCD usa la media aritmética y un múltiplo de la covarianza del subconjunto h
  • Ventajas: robustez frente a outliers, aplicable a alta dimensión
  • Limitaciones: alto costo computacional para datasets grandes, elección de h afecta resultados

Aplicaciones

  • Finanzas: evaluación de riesgos, optimización de carteras
  • Biología: análisis de datos experimentales con observaciones atípicas
  • Ciencias sociales: datos con variabilidad inherente

6. Visualización Multivariada y Mahalanobis

📝 Resumen

Distancia de Mahalanobis

Mide la distancia de un punto a la distribución multivariada, teniendo en cuenta la correlación entre variables y sus escalas diferentes.

D²(M) = (x - μ)ᵀ S⁻¹ (x - μ)
  • S es la matriz de covarianza
  • A diferencia de Euclídea, considera las correlaciones entre variables
  • Es invariable ante transformaciones lineales de las variables
  • Útil para detectar outliers multivariados

Visualización de Datos Multivariados

  • Matrices de dispersión: muestran relaciones bivariadas entre todas las pares de variables
  • Gráficos de perfil: representan observaciones como líneas conectando valores de cada variable
  • Coordenadas paralelas: cada eje representa una variable, cada observación es una línea
  • Gráficos de radar/spider: útiles para comparar perfiles de observaciones

Comparación de Varianzas

  • Para diferencia de proporciones: (p̂₁ - p̂₂) ± z*·SE
  • Para cociente de varianzas: F-distribution

8. Pruebas No Paramétricas

📝 Resumen

Las pruebas no paramétricas no asumen distribución específica de los datos. Son alternativas robustas cuando se violan supuestos de normalidad.

Cuándo usar No Paramétricas

  • Los datos no siguen distribución normal
  • Muestras pequeñas (n < 30)
  • Datos ordinales o con escalas no métricas
  • Presencia de outliers

Alternativas No Paramétricas

ParamétricoNo Paramétrico
t de 2 muestrasWilcoxon-Mann-Whitney (rank-sum)
t pareadoWilcoxon con signos/rangos
ANOVA 1-factorKruskal-Wallis
ANOVA 2 factoresFriedman
Correlación de PearsonSpearman / Kendall
Chi-cuadradoFisher exacto (cuando frecuencias esperadas < 5)

Test de Shapiro-Wilk

  • Contrasta normalidad: H₀: los datos siguen distribución normal
  • Rechazar H₀ si p-valor < α (0.05)
  • Potente para muestras pequeñas

Test de Anderson-Darling

  • Mide qué tan bien siguen los datos una distribución específica
  • Menor estadístico → mejor ajuste
  • H₀: los datos siguen la distribución especificada

9. Análisis de Correspondencias (CA/MCA)

📝 Resumen

Análisis de Correspondencia Simple (CA)

  • Similar a PCA pero para variables categóricas
  • Técnica descriptiva para identificar asociación entre categorías de filas y columnas
  • Reduce dimensionalidad (generalmente 2 ejes principales)
  • Usa la matriz de distancias chi-cuadrado (no varianzas/covarianzas)
  • Los ejes maximizan la inercia (asociación entre filas y columnas)

Supuestos del Test de Independencia

  • n > 50
  • Observaciones independientes
  • Frecuencias esperadas > 5 (máximo 20% con < 5)

Análisis de Perfiles

  • Perfil Fila: proporción de cada categoría de variable j en la fila i
  • Perfil Columna: proporción de cada categoría de variable i en la columna j
  • Si hay independencia, los perfiles serán homogéneos y similares al perfil medio

Análisis de Correspondencia Múltiple (MCA)

  • Extensión del CA para más de 2 variables categóricas
  • Similar al PCA pero para múltiples variables cualitativas
  • Tablero de Burt: tabla de contingencia de todas las combinaciones de categorías

Funciones en R

10. Comparación de más de Dos Poblaciones (ANOVA)

📝 Resumen

Problema de las Comparaciones Múltiples

Si comparamos k grupos con pruebas t por pares, se hacen k(k-1)/2 pruebas. Con α=0.05, el riesgo acumulado de al menos un error Tipo I crece rápidamente.

P(al menos un error) = 1 - (1-α)^(k(k-1)/2)

Ejemplo: con 5 grupos → 10 comparaciones → P ≈ 40% de falso positivo.

Análisis de Varianza (ANOVA)

  • Permite comparar múltiples grupos en un único test, controlando el error Tipo I global
  • H₀: μ₁ = μ₂ = ... = μₖ (todas las medias son iguales)
  • H₁: al menos una media es diferente
  • Se usa el estadístico F de Snedecor

Tipos de ANOVA

  • Un factor (1 criterio): variable independiente categórica (tratamiento)
  • Dos factores (2 criterios): se agrega variable de bloqueo para reducir error experimental

Requisitos

RequisitoTest GráficoTest Analítico
Independencia—Por diseño
NormalidadQQ-plotShapiro-Wilk, Anderson-Darling
Homocedasticidad—Bartlett, Levene

Descomposición de la Varianza

SCT = SCE + SCD
  • SCT: Suma de Cuadrados Total (variación total)
  • SCE: Suma de Cuadrados del Error (variación dentro de grupos)
  • SCD: Suma de Cuadrados del Tratamiento/Diseño (variación entre grupos)
  • Estadístico F = (SCD/gl_D) / (SCE/gl_E)

Pruebas Post-hoc

  • Si ANOVA rechaza H₀, se necesitan pruebas adicionales para saber qué pares difieren
  • Tukey HSD, Bonferroni, Scheffé, etc.

Alternativas No Paramétricas

  • Kruskal-Wallis: alternativa a ANOVA de 1 factor
  • Friedman: alternativa a ANOVA de 2 factores con bloques

11. Regresión Lineal (Simple y Múltiple)

📝 Resumen

Regresión Lineal Simple

Y = β₀ + β₁X + ε
  • Relaciona una variable dependiente (Y) con una independiente (X)
  • Método de Mínimos Cuadrados: minimiza ∑(yᵢ - ŷᵢ)²
  • β₀ = intercepto, β₁ = pendiente

Regresión Lineal Múltiple

Y = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ + ε
  • Usa múltiples variables independientes para explicar Y
  • Modelo Poblacional: Y = β₀ + β₁X₁ + β₂X₂ + ε
  • Modelo Muestral: ŷᵢ = b̂₀ + b̂₁x₁ + b̂₂x₂ + eᵢ
  • Los coeficientes se obtienen resolviendo el Sistema de Ecuaciones Normales de Gauss

Supuestos del Modelo

  • Linealidad entre X e Y
  • Independencia de los residuos
  • Homocedasticidad (varianza constante de residuos)
  • Normalidad de los residuos
  • No multicolinealidad (en regresión múltiple)

Métricas de Bondad de Ajuste

MétricaDescripción
R²Proporción de varianza de Y explicada por el modelo
R² ajustadoR² penalizado por número de predictores
Error estándarDE de los residuos
F globalSignificatividad general del modelo

Correlación

  • Pearson (r): mide relación lineal entre variables continuas, [-1, 1]
  • R² = r² en regresión simple

12. Regresión Logística, SVM, KNN

📝 Resumen

Regresión Logística

  • A pesar del nombre, es un método de clasificación binaria, no predice valores continuos
  • Usa la función sigmoide (logística) para convertir combinación lineal en probabilidad [0,1]
  • Log-odds: ln(p/(1-p)) = β₀ + β₁X₁ + ... + βₖXₖ
  • Umbral de clasificación: p > 0.5 → clase 1, p ≤ 0.5 → clase 0

¿Por qué no usar regresión lineal?

  • La regresión lineal puede predecir valores < 0 o > 1 (imposible para probabilidades)
  • La función logística transforma la combinación lineal al rango [0,1]

Ventajas de Regresión Logística

  • Fácil de implementar e interpretar
  • No supone distribución de clases
  • Rápida en clasificación
  • Los coeficientes indican importancia de características (Odds Ratio)

Desventajas

  • Construye límites lineales de decisión
  • Requiere multicolinealidad baja/media
  • No resuelve problemas no lineales

K-Vecinos Más Cercanos (KNN)

  • Clasifica según las k observaciones más cercanas
  • No tiene fase de entrenamiento (lazy learning)
  • Simple pero sensible a la escala y al valor de k

Máquinas de Vectores de Soporte (SVM)

  • Trazan fronteras de decisión flexibles (incluso no lineales con kernel)
  • Buscan el hiperplano con máxima margen entre clases
  • Flexibles pero menos interpretables

Funciones en R

predict(modelo, type = "response")  # probabilidades

# KNN (librería class)
library(class)

# SVM (librería e1071)

13. Matriz de Confusión y Curva ROC

📝 Resumen

Matriz de Confusión

Predicho 0Predicho 1
Real 0VN (Verdadero Negativo)FP (Falso Positivo)
Real 1FN (Falso Negativo)VP (Verdadero Positivo)

Métricas de Evaluación

MétricaFórmulaInterpretación
Accuracy(VP + VN) / TotalProporción total de aciertos
PrecisiónVP / (VP + FP)De los predichos positivos, cuántos son realmente positivos
Sensibilidad (Recall)VP / (VP + FN)De los realmente positivos, cuántos detecta
EspecificidadVN / (VN + FP)De los realmente negativos, cuántos identifica
F1-Score2 × (Prec × Sens) / (Prec + Sens)Media armónica de Precisión y Sensibilidad

Curva ROC

  • Grafica TVP (Tasa de Verdaderos Positivos = Sensibilidad) vs TFP (Tasa de Falsos Positivos = 1 - Especificidad)
  • Se construye variando el umbral de clasificación
  • Curva perfecta: TVP=1, TFP=0 para todos los umbrales
  • Diagonal: clasificación aleatoria (sin capacidad)
  • Mejor modelo: curva más alejada de la diagonal, cerca del ángulo superior izquierdo

Área Bajo la Curva (AUC)

AUCInterpretación
1.0Modelo perfecto
> 0.8Muy buen desempeño
0.5Clasificación aleatoria (sin capacidad)
< 0.5Peor que aleatorio (error en diseño/datos)

Ventajas del AUC

  • Independiente del umbral de clasificación
  • Facilita comparación entre modelos
  • Menos sensible al desequilibrio de clases

Aplicaciones

  • Medicina: evaluación de pruebas diagnósticas
  • Seguridad informática: detección de intrusiones
  • Marketing: predicción de comportamiento del cliente
